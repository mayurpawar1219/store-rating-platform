# Store Rating Platform

## Tech Stack

Frontend:
- React
- React Router
- Axios

Backend:
- Node.js
- Express.js
- JWT Authentication

Database:
- MySQL

## Features

### Admin
- Dashboard Statistics
- Create Store Owner
- Create Store
- List Users
- List Stores

### User
- Register
- Login
- Search Stores
- Submit Rating
- Update Rating

### Store Owner
- View Ratings
- View Average Rating

## Setup

Backend

npm install
npm run dev

Frontend

npm install
npm run dev
